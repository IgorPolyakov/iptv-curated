# iptv-curated

Курированный IPTV-плейлист, автоматически пересобирается из upstream MEGA-плейлиста.

## Что делает `curate.py`

- фильтрует **RU + мировые топ**, выбрасывает иностранное и взрослое
- группирует по жанрам: Эфирные, Новости, Фильмы, Сериалы, Спорт, Познавательные, Детские, Музыка, Региональные, Прочее RU
- поднимает популярные федеральные каналы вверх
- убирает дубли (макс 2 источника на канал) и битые логотипы

## Обновление

GitHub Action `build-playlist` пересобирает плейлист ежедневно (04:00 UTC) и при каждом изменении `curate.py`. Запустить вручную можно через Actions -> build-playlist -> Run workflow.

upstream: `IPTVRU2026/IPTVMIR`
